# alias
git指令
:::success
ghelp

Git 快捷命令說明：
  gcheck  - 顯示工作目錄和暫存區的狀態 (git status)
  gadd    - 添加文件內容至索引 (git add)
  gcommit - 記錄變更到儲存庫 (git commit -m)
  gpush   - 更新遠端分支 (git push origin master)
使用方法：
  gadd .                    # 添加所有更改
  gadd <file>               # 添加特定文件
  gcommit "提交信息"        # 提交更改
:::
# Usage
```
sbatch model_train.sh # python train.py
sbatch model_test.sh # python test.py
```
